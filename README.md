# digital-restaurant-menu
## Project Overview: <br>
   The Digital Restaurant Menu System is a desktop application developed using Java Swing in NetBeans IDE. It allows users to view and interact with a digital menu displaying food images and prices.
## Technology Used:
* Language: Java
* IDE: NetBeans
* GUI Framework: Swing (JFrame, JPanel, JLabel, JButton, etc.)
## Features:
* Interactive Menu: Displays the food names.
* Clickable Items: When an item name is clicked, it shows an image, and price.
* Image Display: Each food item has a corresponding image for better visualization.
